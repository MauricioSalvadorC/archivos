Seamless integration with GitHub

Join more than 1 million developers already using CircleCI’s integration with GitHub
to optimize your build and test automation.

If you are a GitHub user and not ready for private projects, choose public repos. We ask for read/write access to make
your experience seamless on CircleCI. By signing up, you are agreeing to our SaaS Agreement and Privacy Policy.

Trusted By

No plugins to install

CircleCI mirrors your GitHub team permissions and privileges,
which means you can start building right away.

View your test results in GitHub

CircleCI automatically runs your build and test processes
whenever you commit code, and then displays the build status
in your GitHub branch.

Choose the environment you need

Create a config.yml in your project’s root directory and CircleCI
will read it each time it runs a build.

We are where you are

Communicate build statuses with the tools your teams already
uses like Slack, IRC, and email.
